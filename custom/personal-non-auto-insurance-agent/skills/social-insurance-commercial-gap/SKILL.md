# 社保体系与商保衔接知识库

## Purpose
为gap-diagnosis-engine提供中国多层次医疗保障体系的完整知识，使其能够精确测算社保报销后的个人自费缺口，为商业医疗险/重疾险的保额建议提供量化依据。

## When to Use
- 测算用户医疗报销缺口时
- 比对社保与商保覆盖范围时
- 评估惠民保能否满足用户需求时
- 计算各地医保差异导致的缺口时

## Core Knowledge

### 一、中国多层次医疗保障体系总览

```
第一层：基本医疗保险（全民覆盖，保基本）
├── 城镇职工基本医疗保险（职工医保）
├── 城乡居民基本医疗保险（居民医保，合并新农合）
└── 覆盖95%+人口

第二层：大病保险（基本医保基础上的二次报销）
├── 职工大病保险/居民大病保险
└── 基本医保报销后，自付部分超过起付线的可再报50%-70%

第三层：医疗救助（针对困难群众）
└── 低保/特困/因病致贫家庭的兜底救助

第四层：商业健康保险（自愿购买，补充保障）
├── 百万医疗险
├── 中端/高端医疗险
├── 重疾险（给付型）
├── 惠民保（政府指导的普惠型商业医疗险）
└── 特定疾病保险
```

### 二、基本医保三大目录

#### 1. 药品目录

```
【甲类药品】（临床必需、使用广泛、价格低）
├── 100%纳入医保支付范围
├── 按各地报销比例报销
└── 约1800余种（2023版）

【乙类药品】（可供选择、价格略高）
├── 个人先自付10%-30%，剩余部分再按比例报销
└── 约800余种（2023版）

【丙类药品】（不在医保目录内）
├── 完全自费
├── 含：进口药、靶向药（部分）、新药、保健品等
└── 部分靶向药通过谈判纳入医保后转为乙类
```

#### 2. 诊疗项目目录

```
【全额报销项目】
├── 基本检查检验（血常规、尿常规、X光等）
├── 基本治疗项目（输液、换药等）
└── 基本手术项目

【部分报销项目】
├── 个人先自付一定比例（通常10%-20%）
├── 含：CT、MRI、PET-CT、部分介入治疗
└── 伽玛刀、质子重离子等（各地差异大）

【不予报销项目】
├── 美容整形
├── 非功能性牙科
├── 助孕/辅助生殖
├── 保健体检
└── 器官移植供体费用
```

#### 3. 医疗服务设施目录

```
【可报销】
├── 普通病房床位费（限医保标准内）
├── 基本护理费
└── 普通伙食费（部分）

【不可报销】
├── 特需病房/单人间床位费
├── 空调费/电视费/电话费
├── 护工费（非医疗护理）
└── 交通费/营养费
```

### 三、医保报销计算模型

#### 报销公式

```
实际报销金额 = (总费用 - 自费费用 - 乙类先自付 - 起付线) × 报销比例

其中：
- 总费用 = 全部医疗费用
- 自费费用 = 丙类药品 + 不可报销诊疗项目 + 超标床位费等
- 乙类先自付 = 乙类药品/项目 × 个人先自付比例（10%-30%）
- 起付线 = 各地规定的起付金额
- 报销比例 = 各地规定的报销比例（受封顶线限制）
```

### 四、大病保险（二次报销）

```
【大病保险起付线】
├── 通常为1-2万元（各地不同）
├── 起付线以上：超过部分分段按比例报销
└── 例：北京2024年大病保险起付线30404元

【报销比例】
├── 起付线-5万元：60%
├── 5-10万元：65%
├── 10-15万元：70%
├── 15-20万元：75%
└── 20万元以上：80%

【封顶线】
├── 职工大病保险：通常不设封顶或50万+
├── 居民大病保险：通常30-50万
└── 部分地区上不封顶（如北京）

【大病保险报销计算】
大病报销金额 = (基本医保报销后个人自付 - 大病起付线) × 分段报销比例
```

### 五、惠民保（城市定制型商业医疗险）

```
【典型特征】
├── 保费：通常59-199元/年，统一定价不分年龄
├── 投保门槛：无年龄限制、无健康告知（带病可投保）
├── 既往症：可投但既往症赔付比例降低（通常降至30%）
├── 免赔额：通常1.5-2万（医保内外分开计算）
├── 报销比例：医保内70%-80%，医保外50%-70%
├── 封顶线：100-300万
├── 特定药品：通常含15-35种特药
└── 参保期：限定时间（通常年末1-2个月），错过等一年

【惠民保 vs 百万医疗险对比】
| 维度 | 惠民保 | 百万医疗险 |
|------|--------|-----------|
| 保费 | 59-199元 | 200-2000元 |
| 投保门槛 | 几乎无 | 有健康告知 |
| 免赔额 | 1.5-2万 | 1万 |
| 报销比例 | 50%-80% | 100%（社保后） |
| 续保稳定性 | 不确定（政策驱动） | 较确定 |
| 适合人群 | 老年人/非标体/预算极度有限 | 健康人群/追求全面保障 |
```

### 六、医疗报销缺口测算模型

```
【缺口测算公式】

总医疗费用缺口 = 期望保障水平 - 社保实际报销 - 现有商保报销

其中：
期望保障水平 = 用户期望的医疗费用自付上限
  ├── 基础型：年自付不超过2-3万（对应百万医疗险）
  ├── 标准型：年自付不超过1万（对应中端医疗险）
  └── 品质型：年自付不超过0（对应高端医疗险+0免赔）

【典型场景缺口测算示例】

场景：北京职工，年收入15万，有职工医保，无商保
假设发生：恶性肿瘤治疗，年总费用30万

费用结构：
├── 甲类药品+项目：8万（100%纳入）
├── 乙类药品+项目：10万（先自付15%=1.5万）
├── 丙类/自费（靶向药等）：12万

医保报销：
├── 纳入报销 = 8 + (10-1.5) = 16.5万
├── 报销金额 = (16.5 - 0.13) × 85% ≈ 13.9万
├── 大病保险：(16.5-13.9-3.04) × 60% ≈ 负数 → 未触发
├── 实际自付 = 30 - 13.9 = 16.1万

缺口分析：
├── 百万医疗（1万免赔）：可报销 16.1 - 1 = 15.1万
├── 中端医疗（0免赔+特需）：可报销更多，就医体验更好
├── 重疾险（确诊即赔50万）：自付后用赔付金弥补收入损失
└── 结论：医疗险缺口 = 16.1万/年（重大疾病场景）
```

## Instructions

1. **报销测算步骤**：确定医保类型→获取当地参数→按三大目录拆分费用→逐层计算报销→输出缺口
2. **城市参数匹配**：根据用户所在地市匹配最近似的医保参数
3. **多重保障叠加**：先算基本医保→再算大病保险→再算惠民保→最后算商业医疗险，逐层扣除
4. **缺口分级**：年自付≤1万为安全、1-5万为预警、5万+为严重缺口
5. **数据时效**：各地医保政策每年可能调整，参数库需年度更新
