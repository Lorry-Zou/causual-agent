# 保单管理运营规范

## Purpose
为policy-orchestrator提供标准化的保单运营规则，确保保单全生命周期管理（缴费、预警、批改、贷款、退保、复效）的操作准确性和时效性。

## When to Use
- 定时巡检保单缴费状态时
- 触发宽限期/失效/复效/终止状态变更时
- 受理退保/批改/贷款申请时
- 保费豁免事件触发时
- 生成保单管理报告时

## Core Knowledge

### 一、缴费全生命周期状态机

```
                    ┌──────────────────────────────────────────┐
                    │              NORMAL（正常缴费）              │
                    │  保费按时缴纳，保单处于有效保障状态           │
                    └──────┬──────────────────────┬────────────┘
                           │ 逾期未缴费             │ 缴费成功
                           ▼                       │
                    ┌──────────────────────┐       │
                    │   GRACE（宽限期）      │       │
                    │ 60天内补交保费即可     │───────┘
                    │ 宽限期内发生事故正常赔  │
                    │ 但需扣除欠缴保费       │
                    └──────┬───────────────┘
                           │ 60天届满仍未缴费
                           ▼
                    ┌──────────────────────┐
                    │  LAPSED（失效/中止）   │

                           │

              ▼            │            ▼
    ┌─────────────────┐    │    ┌──────────────────┐

    │ （复效成功）      │    │    │ （终止/不可恢复）   │

                    │ SURRENDERED（退保）   │

                    └──────────────────────┘
```

### 二、关键时间节点详解

#### 1. 宽限期（Grace Period）

```
【法规依据】《保险法》第三十六条
【期限】60天（自缴费对应日/保单周年日起算）
【起算方式】
  ├── 年交/半年交/季交/月交：自约定的缴费日期次日起算
  └── 注：不同保险公司对起算日微有差异，以合同约定为准

【宽限期内的权利义务】
  ├── 保障有效：宽限期内发生保险事故，保险公司仍需赔付
  ├── 扣减欠费：赔付金额中可扣除欠缴保费
  ├── 补缴保费：补缴后保单正常延续，无额外等待期
  └── 无滞纳金：宽限期内无罚息（与复效不同）

【预警触发时间线】
  ├── 缴费日前30天：首次缴费提醒（友好提醒）
  ├── 缴费日前15天：二次提醒（即将到期）
  ├── 缴费日前7天：三次提醒（紧急）
  ├── 缴费日前1天：最后提醒
  ├── 缴费日当天：确认缴费结果
  ├── 逾期第15天：宽限期中提醒（仍有X天）
  ├── 逾期第45天：宽限期倒计时（仅剩15天）
  └── 逾期第59天：最后通牒（明天失效）
```

#### 2. 复效期（Reinstatement Period）

```
【法规依据】《保险法》第三十七条
【期限】2年（自宽限期届满次日/合同效力中止之日起算）
【复效条件】
  ├── 投保人提出复效申请
  ├── 补缴所有欠缴保费 + 利息
  ├── 重新进行健康告知（保险公司有权重新核保）
  ├── 可能需体检（保险公司根据年龄/保额/中止时长决定）
  └── 保险公司审核同意

【复效后的状态】
  ├── 等待期：重新计算（从复效生效日起）
  ├── 自杀条款：重新计算2年免责期
  ├── 不可抗辩条款：重新计算2年
  └── 保单现价：从中止前现价基础上接续

【复效利息参考】
  ├── 通常为年复利5%-6%或参考同期贷款利率
  └── 具体以合同约定为准
```

#### 3. 犹豫期（Cooling-off Period）

```
【法规规定】
  ├── 长期人身保险：不少于15天（自签收保单次日起算）
  ├── 部分地区/公司/产品：20天（如部分互联网长期险）
  └── 一年期及以下产品：通常无犹豫期

【犹豫期退保流程】
  ├── 投保人书面/在线申请
  ├── 退还全部已缴保费（扣除不超过10元工本费）
  ├── 合同解除，双方恢复原状
  └── 保险公司在收到申请后X个工作日内退款

【实务注意事项】
  ├── 犹豫期最后一天为法定节假日则顺延
  ├── 如进行了体检/核保，退工本费上限10元
  └── 犹豫期内如已发生理赔，通常不可退保
```

#### 4. 保费豁免

```
【豁免类型】
  ├── 投保人豁免：投保人身故/全残/重疾/轻症 → 免交后续保费
  ├── 被保险人豁免：被保险人重疾/轻症/中症 → 免交后续保费
  └── 双豁免：投保人+被保险人双重豁免（常见于夫妻互保）

【豁免触发条件】
  ├── 身故豁免：投保人身故
  ├── 全残豁免：投保人达到全残标准
  ├── 重疾豁免：确诊合同约定的重大疾病
  ├── 轻症豁免（部分产品）：确诊合同约定的轻症
  └── 失能豁免（少量产品）：因意外或疾病丧失工作能力

【豁免后的保单状态】
  ├── 保单继续有效，保障不受影响
  ├── 后续所有应缴保费视同已缴
  ├── 豁免保费不影响保单现金价值增长
  └── 如有附加险，附加险是否豁免需单独确认
```

### 三、保单贷款规则

```
【贷款条件】
  ├── 保单具有现金价值
  ├── 保单有效（非失效/终止状态）
  └── 通常需保单生效满1-2年

【贷款额度】
  ├── 最高可贷：现金价值的80%
  └── 最低贷款：通常1000元起

【贷款利率】
  ├── 通常为央行同期贷款基准利率上浮
  ├── 半年结算一次利息
  └── 参考范围：年化5%-7%

【本息偿还】
  ├── 贷款期限：通常6个月（可续贷）
  ├── 续贷：到期可申请续贷，结清利息即可
  └── 到期未还：利息滚入本金重新计息

【贷款与保单状态】
  ├── 贷款期间保单保障不受影响
  ├── 如发生理赔：赔付金额 = 应赔金额 - 贷款本息
  └── 本息超过现金价值时：保单效力中止
```

### 四、退保处理规则

```
【退保金额】
  ├── 犹豫期内：退还全部保费（扣不超10元工本费）
  ├── 犹豫期后：退还现金价值
  │   ├── 前期现价低：首年可能仅退10%-30%
  │   ├── 现金价值随时间增长逐步接近甚至超过已缴保费
  │   └── 具体看产品：年金/增额寿回本较快，重疾/医疗险较慢
  └── 已发生理赔的保单：通常不可退保或退还金额另计

【退保审核】
  ├── 投保人身份验证
  ├── 银行卡信息核实（退费账户需为投保人本人）
  ├── 是否涉及贷款未结清
  └── 是否有未决理赔

【退保风险提示义务】
  ├── 明确告知退保金额（现金价值）
  ├── 提醒保障中断风险
  ├── 提醒重新投保可能面临的条件变化（年龄增长/健康状况变化）
  └── 建议慎重决策
```

### 五、保单批改操作

```
【可批改项目】
  ├── 受益人变更（需经被保险人同意）
  ├── 联系方式变更（手机/地址/邮箱）
  ├── 缴费账户变更
  ├── 职业类别变更（需重新评估风险等级）
  ├── 投保人变更（需经被保险人同意）
  ├── 保额调整（增加需核保）
  └── 附加险增减

【受益人变更特别注意】
  ├── 需被保险人书面同意（人身保险）
  ├── 可指定顺序和份额
  ├── 变更生效时间：保险公司收到申请并批注后
  └── 未指定受益人：按法定继承顺序
```

## Instructions

1. **状态巡检频率**：
   - 正常保单：每月巡检1次
   - 宽限期保单：每3天巡检1次
   - 失效待复效保单：每月巡检1次
   - 即将到期保单：到期前30天起每天巡检

2. **事件驱动触发**：缴费成功/失败、宽限期到期、复效申请、退保申请等事件即时处理

3. **日志记录**：所有状态变更必须记录时间戳+操作人/触发条件+变更前后状态

4. **异常告警**：大规模缴费失败、保单集中到期等异常情况需触发告警
