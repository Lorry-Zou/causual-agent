# 保险精算与保额测算模型

## Purpose
为gap-diagnosis-engine提供科学的保额测算方法论，使缺口诊断从"凭经验推荐"升级为"数据驱动测算"，为每个用户输出可量化、可解释的保额建议。

## When to Use
- 计算重疾险建议保额时
- 测算家庭寿险保额需求时
- 估算教育金和养老金储蓄目标时
- 用户画像更新后重新校准保额建议时

## Core Knowledge

### 一、重疾险保额测算模型

#### 1.1 三层费用结构

```
重疾保额 = 治疗费用 + 收入损失补偿 + 康复费用

┌─────────────────────────────────────────────────┐
│ 第一层：治疗费用（直接医疗支出）                     │
├─────────────────────────────────────────────────┤
│ 【不同城市等级参考（恶性肿瘤为例）】                  │
│ 一线城市（北上广深）：30-50万                       │
│   ├── 手术费：5-15万                              │
│   ├── 住院费+药费：10-20万                         │
│   ├── 靶向药/免疫治疗：10-20万/年                   │
│   └── 放化疗：5-10万                              │
│                                                   │
│ 二线城市（省会/计划单列市）：20-35万                  │
│ 三四线城市：15-25万                                │
│                                                   │
│ 【不同重疾病种参考】                                │
│ ├── 恶性肿瘤：20-50万（视分型和治疗方式）            │
│ ├── 急性心肌梗死：10-20万（支架/搭桥）               │
│ ├── 脑中风后遗症：15-30万（含长期康复）              │
│ ├── 终末期肾病：10-15万/年（透析）+ 30-50万（移植）  │
│ └── 重大器官移植：30-60万                          │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ 第二层：收入损失补偿                                │
├─────────────────────────────────────────────────┤
│ 重疾后通常需要3-5年康复期                           │
│                                                   │
│ 收入损失 = 年收入 × 康复年限                        │
│                                                   │
│ 【康复年限建议】                                   │
│ ├── 恶性肿瘤：3-5年                               │
│ ├── 心脑血管：2-3年                               │
│ ├── 器官移植：2-5年                               │
│ └── 保守建议：取5年（覆盖最严重情况）                │
│                                                   │
│ 【调整因子】                                      │
│ ├── 家庭经济支柱：×1.2-1.5（家庭依赖度高）          │
│ ├── 双收入家庭：×0.7-1.0（有配偶收入缓冲）          │
│ └── 被动收入充足：×0.3-0.7（租金/理财等）           │
└─────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────┐
│ 第三层：康复费用（非医疗但必需）                     │
├─────────────────────────────────────────────────┤
│ ├── 营养费：1000-3000元/月 × 12-36个月              │
│ ├── 护理费：3000-8000元/月 × 12-36个月（如需）      │
│ ├── 康复器具/无障碍改造：2-10万（一次性）            │
│ ├── 交通食宿（异地就医）：5-15万                    │
│ └── 合计参考：10-30万                             │
└─────────────────────────────────────────────────┘
```

#### 1.2 重疾保额简化公式

```
标准公式：
重疾保额 = 基础治疗费(按城市等级) + 年收入 × 5 + 康复费用(15万)

示例计算：
├── 一线城市白领，年收入20万
├── 治疗费：35万（一线城市均值）
├── 收入损失：20万 × 5 = 100万
├── 康复费用：15万
├── 建议保额：150万
└── 实际配置建议：50-100万重疾险 + 百万医疗险（覆盖治疗费）

分档建议：
├── 极简版（刚需）：30万（覆盖基本治疗）
├── 标准版：50万（覆盖治疗 + 1-2年收入）
├── 全面版：100万+（覆盖治疗 + 3-5年收入 + 康复）
└── 注意：医疗险可覆盖治疗费，重疾险更多补偿收入损失
```

### 二、寿险保额测算模型（家庭负债法）

#### 2.1 需求法公式

```
寿险保额 = 负债覆盖 + 收入替代 + 未来责任

┌──────────────────────────────────────────────┐
│ 负债覆盖                                      │
├──────────────────────────────────────────────┤
│ ├── 房贷余额                                  │
│ ├── 车贷余额                                  │
│ ├── 其他消费贷余额                            │
│ └── 信用卡欠款                                │
└──────────────────────────────────────────────┘
                    +
┌──────────────────────────────────────────────┐
│ 收入替代（家庭年支出 × 替代年限）                │
├──────────────────────────────────────────────┤
│ ├── 家庭年基础支出（衣食住行）                  │
│ ├── 替代年限：最小子女至22岁 or 10-15年        │
│ └── 公式：年支出 × (最小子女至22岁的年数, 最少10年)│
└──────────────────────────────────────────────┘
                    +
┌──────────────────────────────────────────────┐
│ 未来责任                                      │
├──────────────────────────────────────────────┤
│ ├── 子女教育金：50-100万/孩（国内大学）         │
│ │                 100-300万/孩（海外留学）     │
│ ├── 父母赡养费：年赡养费 × 预期赡养年限         │
│ │               通常10-30万/年 × 10-20年      │
│ └── 丧葬及其他善后费用：5-10万                 │
└──────────────────────────────────────────────┘
                    -
┌──────────────────────────────────────────────┐
│ 现有可动用资产                                │
├──────────────────────────────────────────────┤
│ ├── 现有储蓄/投资                             │
│ ├── 现有寿险保额                              │
│ ├── 社保丧葬抚恤金                            │
│ └── 配偶收入折现（如果家庭接受）               │
└──────────────────────────────────────────────┘
```

### 三、教育金储蓄测算

```
【教育金需求（国内）】
├── 幼儿园：3-8万/年 × 3年 = 9-24万
├── 小学-初中：义务教育，主要是课外/兴趣班 2-5万/年 × 9年
├── 高中：学费+生活费 3-8万/年 × 3年 = 9-24万
├── 大学：学费+生活费 5-10万/年 × 4年 = 20-40万
└── 合计（大学阶段为主）：30-50万/孩

【教育金需求（海外本科）】
├── 美国：30-60万/年 × 4年 = 120-240万
├── 英国：25-50万/年 × 3年 = 75-150万
├── 澳洲：20-40万/年 × 3-4年 = 60-160万
└── 亚洲（日韩新马）：10-25万/年 × 3-4年 = 30-100万

【储蓄倒推公式】
所需年储蓄 = 目标金额 × (1 + 通胀率)^(储蓄年限) ÷ 储蓄年限

示例：
├── 孩子3岁，目标国内大学40万（15年后）
├── 通胀后需求 ≈ 40 × (1.03^15) ≈ 62万
├── 若通过年金险储蓄，年缴保费 ≈ 62 ÷ 15 ≈ 4.1万
└── 或通过增额寿：可根据IRR反算
```

### 四、养老储蓄测算

```
【退休收入替代率目标】
├── 国际标准：退休后收入 = 退休前收入的70%-80%
├── 中国社保养老金替代率：约40%-50%（且趋势下降）
├── 缺口 = 目标替代率 - 社保替代率 ≈ 30%-40%
└── 需个人额外储备

【养老金缺口测算公式】
年养老金缺口 = (退休前年收入 × 目标替代率) - 预计社保养老金 - 其他被动收入

【储蓄目标倒推】
总养老金缺口 = 年缺口 × 预期退休后生存年限（通常按至85-90岁）

示例：
├── 35岁，年收入20万，目标退休年龄60岁
├── 退休前年收入（增长后）：约35万
├── 目标年退休收入：35 × 70% = 24.5万
├── 预计社保：35 × 40% = 14万
├── 年缺口：24.5 - 14 = 10.5万
├── 退休后生存25年 → 总缺口 ≈ 262万
├── 25年储蓄期 → 年储蓄（含投资回报）≈ 8-10万
└── 建议配置：养老年金+增额终身寿组合
```

### 五、保额-保费预算约束

```
【保险预算指南】
├── 家庭年保费 ≤ 家庭年收入的 10%-15%
├── 保障型（医疗+重疾+意外+寿险）≤ 10%
├── 理财型（年金+增额寿+教育金）≤ 5%
└── 如预算紧张，优先保障型，理财型暂缓

【保障优先级排序】
1. 医疗险（所有人，第一优先级）
2. 意外险（所有人，低保费高杠杆）
3. 重疾险（家庭支柱优先）
4. 定期寿险（有负债/有家庭责任者）
5. 家财险（有自有住房者）
6. 养老年金（35岁以上）
7. 教育年金/增额寿（有余力者）
```

## Instructions

1. **保额测算顺序**：先测算医疗缺口（医疗险）→ 重疾缺口 → 寿险缺口 → 教育/养老储蓄缺口
2. **城市等级匹配**：一线/二线/三四线使用不同治疗费用基准
3. **家庭结构敏感**：单身/已婚/有孩/多孩/赡养老人 → 不同系数
4. **动态更新**：用户收入变化、房贷变化、家庭成员变化时重新测算
5. **输出格式**：每个险种的建议保额+测算依据+分档方案
