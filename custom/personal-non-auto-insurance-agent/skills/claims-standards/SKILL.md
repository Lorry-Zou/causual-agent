# 理赔标准库

## Purpose
为gap-diagnosis-engine和churn-recovery-agent提供全面的理赔流程与标准知识，支持方案生成时的"可赔性评估"以及流失归因时的"理赔体验分析"。

## When to Use
- gap-diagnosis-engine生成保障方案时评估各险种的实际理赔可行性
- churn-recovery-agent判定"理赔体验差导致流失"时
- 评估用户已有保障的实际保障效力时
- 理赔指引场景（保单出险时）

## Core Knowledge

### 一、各险种理赔条件速查

#### 1. 医疗险理赔

```
【理赔条件】
├── 等待期后发生的疾病（通常30天）
├── 意外导致的医疗无等待期
├── 必须在二级及以上公立医院普通部就医
└── 住院/特殊门诊/门诊手术/住院前后门急诊

【报销公式】
赔款 = (总费用 - 社保报销 - 其他商保报销 - 免赔额) × 赔付比例

├── 社保报销后：100%赔付（限社保范围内费用的100%）
├── 未经社保报销：通常60%赔付
├── 社保范围外费用：视产品而定（百万医疗大多不含，中高端含）
└── 免赔额：1万元（绝对免赔/相对免赔因产品而异）

【不赔范围（高发拒赔场景）】
├── 既往症及其并发症
├── 等待期内出险
├── 未达到免赔额（总费用<社保报销+1万）
├── 就医机构不符合要求（一级医院/私立/诊所）
├── 美容/整形/牙科/眼科屈光
├── 生育相关费用
├── 康复/疗养/保健
├── 非合理且必要的治疗（过度医疗）
└── 未如实告知导致的关联疾病
```

#### 2. 重疾险理赔

```
【理赔触发条件】
├── 确诊即赔：恶性肿瘤（需病理报告）、严重Ⅲ度烧伤等
├── 实施约定手术：冠状动脉搭桥术、重大器官移植等
├── 达到约定状态：终末期肾病（透析90天）、深度昏迷（96小时）等
└── 持续约定天数：脑中风后遗症（180天后仍遗留障碍）等

【理赔材料】
├── 理赔申请书
├── 被保险人身份证件
├── 诊断证明（二甲及以上医院）
├── 病理报告/手术记录/出院小结
├── 相关检查报告（CT/MRI/心电/化验）
└── 银行卡信息

【拒赔高发场景】
├── 等待期内出险（90-180天）
├── 未达到条款约定的疾病状态标准
│   └── 如：脑中风未满180天或未达后遗症标准
├── 投保前未如实告知的既往症相关
├── 免责条款内情况（酒驾/吸毒/故意自伤等）
├── 属于轻症范围而非重疾范围（如原位癌）
└── 超过诉讼时效（2年，自知悉之日起）
```

#### 3. 意外险理赔

```
【理赔条件】
├── 突发的、外来的、非本意的、非疾病的
├── 职业与投保时一致
└── 事故发生在保险期间内

【伤残评定标准】
《人身保险伤残评定标准》（行业标准，1-10级）
├── 1级：赔付100%保额（如：双目永久失明、双肢缺失）
├── 2级：赔付90%
├── ...
├── 10级：赔付10%（如：单耳听力损失≥56dB）
└── 多处伤残：取最高级别 + 叠加（有上限）

【拒赔高发场景】
├── 职业类别不符（投保时为1-2类，出险时从事高危活动）
├── 不属于"意外"（猝死、中暑、高原反应、过敏）
├── 高风险运动（攀岩/跳伞/潜水，除非特别承保）
├── 酒后驾驶/无证驾驶
├── 打架斗殴/故意自伤
└── 战争/核辐射/恐怖袭击（部分产品）
```

#### 4. 寿险理赔

```
【理赔条件】
├── 身故（任何原因，含疾病和意外）
├── 全残（多数定期寿险含全残责任）
└── 保险期间内

【理赔材料（身故）】
├── 理赔申请书（受益人填写）
├── 死亡证明（医学死亡证明+户籍注销证明）
├── 受益人身份证件+关系证明
├── 保险合同
├── 事故证明（如交通事故认定书、工伤认定书）
└── 银行卡信息

【免责条款（法定3条）】
├── 投保人对被保险人故意杀害
├── 被保险人故意犯罪或抗拒刑事强制措施
└── 保险合同成立2年内自杀（无民事行为能力人除外）

【理赔时效】
├── 出险通知：知道事故后10天内
├── 理赔申请：材料齐全后30天内做出核定
├── 赔付：达成协议后10天内支付
└── 诉讼时效：2年（自知悉之日起）
```

### 二、通用理赔流程

```
出险 → 报案 → 提交材料 → 审核 → 核赔决定 → 赔付/拒赔

【各环节详解】

1. 报案（出险通知）
   ├── 时限：通常要求10天内通知保险公司
   ├── 方式：电话/APP/官微/代理人
   └── 提供：保单号+出险人+出险时间+事故简述

2. 提交材料
   ├── 线上：APP/小程序拍照上传
   ├── 线下：邮寄或柜面递交
   └── 补材料：5个工作日内通知补充

3. 审核
   ├── 初审：材料完整性检查（1-3个工作日）
   ├── 调查：复杂案件启动调查（面访/医院核查/公安等）
   └── 调查时限：通常30天内

4. 核赔决定
   ├── 正常赔付：发送理赔决定通知书+赔款
   ├── 部分赔付：部分免责或按比例
   ├── 拒赔：出具拒赔通知书（需说明理由）
   └── 协议赔付：双方协商

5. 赔付时效
   ├── 简单案件：3-7个工作日
   ├── 复杂案件：30天内完成核定
   ├── 达成协议后：10天内支付赔款
   └── 逾期：按银行同期利率支付滞纳金
```

### 三、拒赔原因分类体系

```
【拒赔原因分类（用于流失归因分析）】

A类：投保端问题（可预防）
├── A1：未如实告知（带病投保/隐瞒既往症）
├── A2：职业类别不符（投保时低报职业风险）
├── A3：投保人对条款理解有误（销售误导/自误解）
└── A4：未及时续保导致保单失效

B类：条款端问题（预先存在）
├── B1：属于免责条款范围
├── B2：未达到理赔条件（如重疾未达状态标准）
├── B3：等待期内出险
└── B4：不在保障范围内（如意外险不赔疾病）

C类：就医端问题（可引导）
├── C1：就医机构不符合要求（去了私立/一级医院）
├── C2：未用社保先结算（导致报销比例降低）
├── C3：病历书写问题（主诉不当导致被认定为既往症）
└── C4：未留存必要票据/材料

D类：保险公司端问题（服务问题）
├── D1：理赔审核过慢（超过法定时限）
├── D2：无理拒赔（经投诉/诉讼后改判赔付）
├── D3：理赔金额争议（赔付金额与预期差距大）
└── D4：服务态度差/沟通不畅
```

### 四、就医范围限制速查

| 险种 | 就医范围 | 例外情况 |
|------|---------|---------|
| 社保/基本医保 | 定点医院 | 急诊可就近 |
| 百万医疗险 | 大陆二级及以上公立医院普通部 | 部分可含指定私立 |
| 中端医疗险 | 大陆二级及以上（含特需/国际部） | 部分含指定私立 |
| 高端医疗险 | 全球/全球除美/大中华 | 含昂贵医院 |
| 意外险（医疗） | 二级及以上公立医院 | 紧急可就近 |
| 重疾险（确诊） | 二级及以上医院 | 通常不限公立/私立 |
| 旅游险 | 境内/境外指定 | 以保单约定为准 |

## Instructions

1. **方案评估时**：检查推荐产品是否与用户常就医区域/偏好匹配（如用户习惯去私立，则百万医疗不适用）
2. **流失归因时**：历史理赔记录按A/B/C/D分类体系归类，判定流失是否与理赔体验相关
3. **拒赔正当性判断**：对比拒赔理由与条款约定，判断拒赔是否合理
4. **数据更新**：理赔标准以保险公司最新条款为准，年度更新
