# 保险行业合规规则库

## Purpose
为policy-orchestrator内置合规审核能力，确保所有面向用户的输出（续保提醒话术、批改指引、保单报告）符合银保监（现国家金融监督管理总局）合规要求，避免违规风险。

## When to Use
- 生成任何面向用户的文字内容时（话术合规过滤）
- 保单批改/退保/理赔指引时
- 利益演示和产品说明时
- 判断销售行为是否合规时

## Core Knowledge

### 一、话术合规红线（销售禁语清单）

#### 🚫 严禁表述（零容忍）

| 禁语类别 | 禁语示例 | 合规替代方案 |
|---------|---------|------------|
| 承诺收益 | "保证收益""稳赚不赔""保本保息" | "历史结算利率为X%，未来收益以实际为准" |
| 夸大保障 | "什么都赔""买了就放心" | "具体保障范围以条款约定为准，建议您仔细阅读" |
| 虚假承诺理赔 | "一定能赔""理赔包在我身上" | "符合条款约定的情况均可申请理赔" |
| 饥饿营销 | "最后一天""再不买就没了""限时抢购" | "该产品目前处于在售状态" |
| 贬低同业 | "XX公司理赔差""那个产品不如我们" | "不同产品各有特点，建议您根据自身需求选择" |
| 诱导隐瞒 | "这个不用说""不用告知也能赔" | "请如实填写健康告知，这是理赔的基础" |
| 代签代办 | "我帮你填就行了" | "请本人如实填写" |
| 返佣暗示 | "买了可以返你XX" | 严禁任何形式的返佣承诺 |

#### ⚠️ 谨慎表述（需附加说明）

| 表述 | 需附加的说明 |
|------|------------|
| "高收益" | 需注明非保证收益，并展示低/中/高三档演示 |
| "全面保障" | 需注明"具体保障范围以保险条款为准" |
| "有病也能买" | 需说明核保可能产生除外/加费结论 |
| "保证续保" | 需明确"保证续保期间为XX年"，不得笼统表述 |

### 二、利益演示规范

#### 分红险/万能险/投连险利益演示要求：

```
【必须同时展示三档（投连险除外）】
├── 低档：保证利益 + 非保证利益(低档)
├── 中档：保证利益 + 非保证利益(中档，4.5%演示)
└── 高档：保证利益 + 非保证利益(高档，6%演示)

【必须包含的声明】
├── "该利益演示基于公司精算假设，不代表未来实际收益"
├── "非保证利益部分具有不确定性"
└── "本演示仅为帮助您理解产品之用"

【演示周期】
├── 需覆盖整个保险期间的关键时间节点
└── 通常展示：第5/10/15/20/30年及领取期各年
```

#### 增额寿利益演示要点：
```
├── 必须展示：现金价值表（明确保证部分）
├── 有效保额递增轨迹
├── 减保/部分退保对现金价值的影响
└── 强调：现金价值超过已缴保费的时间点（回本周期）
```

### 三、双录（录音录像）操作要点

```
【需要双录的场景】（根据银保监规定）
├── 60岁以上投保人购买一年期以上人身保险
├── 通过保险兼业代理机构销售一年期以上人身保险
├── 销售投资连结保险
└── 向60岁以上投保人销售（部分地区规定）

【双录的关键环节】
├── 1. 出示身份证明（销售人员+投保人）
├── 2. 说明产品名称、保险期间、缴费期间、缴费金额
├── 3. 说明保险责任（逐项说明）
├── 4. 说明责任免除条款（逐条说明）
├── 5. 说明犹豫期及退保损失
├── 6. 投保人明确表示自愿投保
├── 7. 健康告知逐项确认（如有）
└── 8. 投保人签名确认

【双录存储要求】
├── 保存期限：不少于5年（保险合同终止日起算）
├── 影像清晰度：可辨识面部特征
└── 声音清晰度：可听清对话内容
```

### 四、异地投保规则

```
【原则】互联网保险业务不得将营销活动拓展至未设分支机构的区域（部分放开）

【分类规则】
├── 互联网专属产品：可全国销售（需在产品备案时标注）
├── 线下产品：原则上仅在设立分支机构的省/市销售
├── 意外险/旅游险：通常全国可售
└── 医疗险/重疾险：需确认当地是否有认可医疗机构

【实务注意】
├── 投保地址：以投保人实际居住/工作地为准
├── 批改地址：不得通过虚填地址规避异地限制
└── 就医范围：须与承保地确认异地就医是否可理赔
```

### 五、既往症核保常识

```
【既往症定义（《保险法》司法解释三）】
指投保前已患的被保险人已知或应当知道的疾病或症状

【核保结论分类】
├── 标准体：不影响承保，标准费率
├── 次标准体-加费：增加保费承保（如+25%/+50%）
├── 次标准体-除外：不保障该既往症相关疾病
├── 延期：暂时不能承保，观察一段时间后重新评估
└── 拒保：无法承保

【常见体检异常的核保倾向】
| 异常项 | 医疗险 | 重疾险 | 定期寿险 |
|--------|--------|--------|----------|
| 甲状腺结节TI-RADS 1-2级 | 除外或标准体 | 标准体或除外 | 通常标准体 |
| 甲状腺结节TI-RADS 3级 | 除外 | 除外或加费 | 通常标准体 |
| 甲状腺结节TI-RADS 4级+ | 拒保/延期 | 拒保/延期 | 可能加费 |
| 乳腺结节BI-RADS 1-2级 | 除外或标准体 | 标准体 | - |
| 乳腺结节BI-RADS 3级 | 除外 | 除外或加费 | - |
| 高血压1级（<160/100） | 可能加费 | 加费 | 标准体或微调 |
| 高血压2级+（≥160/100） | 拒保 | 拒保/延期 | 加费或拒保 |
| 血糖异常（空腹>7.0） | 拒保 | 拒保 | 可能加费 |
| BMI>30 | 可能加费 | 加费 | 标准体或微调 |
| BMI>35 | 拒保 | 拒保/延期 | 加费或拒保 |
| 脂肪肝（轻度） | 标准体或微调 | 标准体 | 标准体 |
| 脂肪肝（中重度/肝功能异常） | 加费/延期 | 加费/延期 | 加费 |
```

### 六、信息披露要求

```
【投保时必须披露的信息】
├── 条款全文（电子版/纸质版）
├── 产品说明书（一年期以上产品）
├── 投保提示书
├── 健康告知书
├── 免责条款说明
└── 犹豫期权益说明

【事后信息披露】
├── 保单年度报告（部分产品）
├── 红利通知书（分红险，每年）
├── 万能险结算利率公告（每月）
└── 缴费提醒（到期前）
```

## Instructions

1. **话术过滤流程**：所有面向用户的输出文本 → 扫描禁语/敏感词 → 按合规替代方案修正 → 输出
2. **利益演示核查**：检查是否包含三档演示+免责声明+非保证利益提示
3. **双录触发判断**：根据投保人年龄+产品类型判断是否需要双录提示
4. **异地校验**：投保地址与产品备案区域比对
5. **合规优先级**：合规规则 > 营销效果，任何情况不得为了转化率违反合规底线
