# 险种条款与投保规则

## Purpose
为policy-orchestrator提供全险种详细的投保规则和条款约束，确保保单录入校验、批改合规检查、续保判断和风险预警的准确性。

## When to Use
- 新保单录入时校验用户是否符合投保条件
- 保单批改时检查合规性约束
- 续保/到期预警时判断续保可能性
- 保单状态变更时依据条款规则处理

## Core Knowledge

### 二、各险种核心条款要点

#### 1. 医疗险关键条款

```
【保障责任】
├── 一般医疗保险金：住院医疗+特殊门诊+门诊手术+住院前后门急诊
├── 重疾医疗保险金（部分产品）：恶性肿瘤等重疾0免赔
├── 质子重离子治疗（部分产品）：100%报销
└── 特定药品费用（部分产品）：靶向药/免疫药

【免责条款（常见）】
├── 既往症及其并发症（核心免责）
├── 美容整形、牙科治疗（非意外）
├── 生育相关费用
├── 高风险运动导致的意外
├── 违法犯罪导致的伤害
└── 等待期内出险

【续保规则】
├── 非保证续保：每年需审核，停售风险
├── 阶段性保证续保：6年/20年内保证续保
└── 保证续保期间内：不因理赔和健康状况拒绝续保
```

#### 2. 重疾险关键条款

```
【疾病定义（监管规定28种必保重疾+3种必保轻症）】
├── 确诊即赔型：恶性肿瘤、严重Ⅲ度烧伤、多个肢体缺失
├── 实施了约定手术：冠状动脉搭桥术、重大器官移植
├── 达到约定状态：终末期肾病（透析90天）、深度昏迷（96小时）
└── 持续约定天数：脑中风后遗症（180天）

【赔付规则】
├── 单次赔付：赔付后合同终止
├── 分组多次赔付：每组限赔1次，组间有间隔期（通常180天）
├── 不分组多次赔付：任意重疾可赔多次，间隔期通常365天
└── 轻症/中症：赔付比例20%-60%基本保额，多为额外给付

【免责条款】
├── 投保前已患疾病（不如实告知）
├── 等待期内出险（通常退保费）
├── 故意自伤/自杀（2年内）
├── 吸毒/酒驾/无证驾驶
└── 战争/核辐射
```

#### 3. 意外险关键条款

```
【保障责任】
├── 意外身故：100%保额
├── 意外伤残：按《人身保险伤残评定标准》1-10级赔付（100%-10%）
├── 意外医疗：报销制，通常100免赔+100%报销（限社保内）
├── 猝死（部分产品）：通常赔付20%-50%保额
└── 交通意外额外赔付：航空/火车/轮船/汽车叠加赔付

【职业分类与费率】
├── 1-2类：办公室人员，标准费率
├── 3类：轻度体力/外勤，标准费率或微调
├── 4类：中度体力/操作工，加费20%-50%
├── 5类：重度体力/高空作业，加费100%-200%或拒保
└── 6类及特殊：拒保（需专项产品）
```

#### 4. 定期寿险关键条款

```
【保障责任】
├── 身故保险金：100%保额
├── 全残保险金（多数含）：100%保额
└── 部分产品含：交通意外身故额外赔付

【免责条款（通常3条法定免责）】
├── 投保人对被保险人的故意杀害
├── 被保险人故意犯罪或抗拒刑事强制措施
└── 被保险人2年内自杀（无民事行为能力人除外）

【缴费与宽限期】
├── 宽限期：60天（自缴费对应日起）
├── 复效期：2年（宽限期后保单中止可申请复效）
└── 复效条件：补交欠缴保费+利息+健康告知重新审核
```

#### 5. 年金险关键条款

```
【领取规则】
├── 领取起始年龄：55/60/65/70岁（女性通常早于男性5年）
├── 领取频率：年领（100%基本保额）或月领（8.5%基本保额）
├── 保证领取期：通常20年或至80岁
│   └── 保证领取期内身故：剩余未领年金一次性给付受益人
└── 终身领取：活多久领多久

【现金价值与退保】
├── 前期现价低：前5-10年退保损失大
├── 回本周期：通常10-15年现金价值超过已缴保费
└── 退保金：退还保单现金价值
```

#### 6. 增额终身寿险关键条款

```
【保额递增规则】
├── 有效保额 = 基本保额 × (1 + 递增比例)^(保单年度-1)
├── 递增比例：3.5%-4.0%（定价利率调整后降至约3.0%）
└── 身故给付 = max(有效保额, 现金价值, 已缴保费×给付系数)

【灵活领取（减保/部分退保）】
├── 减保：每年不超过基本保额的20%
├── 保单贷款：可贷现金价值的80%，利率约5%-6%
└── 减额交清：停止缴费转为减额保单
```

### 三、缴费与状态管理规则

```
【缴费方式】
├── 趸交：一次性缴清
├── 年交：每年一次，通常可享2%-5%优惠
├── 半年交：年缴保费×0.52
├── 季交：年缴保费×0.265
└── 月交：年缴保费×0.09

【缴费状态机】
正常缴费 → (逾期60天) → 宽限期 → (60天后仍未缴费) → 中止期/失效
                                                    ↓
                                          (2年内申请复效)
                                                    ↓
                                          复效成功 → 正常缴费
                                                    ↓
                                          (2年内未复效) → 终止（不可恢复）
```

### 四、宽限期详细计算规则

```
宽限期起算日 = 保单周年日/缴费对应日
宽限期截止日 = 起算日 + 60天

示例：
- 缴费对应日: 2024-03-15
- 宽限期: 2024-03-15 ~ 2024-05-14（共60天）
- 宽限期内出险：正常赔付，但需扣除欠缴保费
- 宽限期后：保单失效，出险不赔付
- 复效窗口：2024-05-15 ~ 2026-05-14
```

## Instructions

1. **保单录入校验**：按险种-年龄-职业-健康告知逐项校验
2. **缴费提醒节点**：到期前30天/15天/7天/当天/逾期30天/逾期60天
3. **续保评估**：检查年龄是否超限、理赔记录、产品是否停售
4. **状态变更**：严格按条款约定时间节点触发状态变更
5. **条款冲突检测**：同一被保险人多保单时检测是否有条款冲突
