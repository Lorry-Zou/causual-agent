### 三、预算分配黄金比例

```

```

### 五、方案预算约束检测

```
【预算检测规则】
├── 年总保费 > 家庭年收入 × 20% → ⚠️ 预算过重警告
├── 年总保费 < 家庭年收入 × 5% → ℹ️ 保障可能不足
├── 保障型保费 > 理财型保费 → ✅ 健康（先保障后理财）
└── 理财型保费 > 保障型保费 → ⚠️ 不合理（保障不足时不应过度理财）

【预算优化策略】
├── 优先配置医疗+意外（低保费高杠杆）
├── 重疾险可选消费型降低预算
├── 定期寿险替代终身寿险
├── 拉长缴费期降低年缴压力
└── 家庭保单组合购买享受折扣
```

## Instructions

1. **方案生成流程**：确定用户画像→评估各险种需求→匹配产品类型→计算年保费→预算约束检测→输出三档方案
2. **费率动态更新**：产品费率以保险公司实际定价为准，应支持手动更新费率表
3. **预算硬约束**：如果最低档方案超出用户预算，按优先级裁减（先裁理财型，再降重疾保额，最后调整寿险）
4. **非标体标记**：方案输出时自动检测用户健康标签，非标体方案自动替换为核保友好的产品类型
